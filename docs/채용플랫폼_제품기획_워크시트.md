- 핵심 고객: 인스타·유튜브·틱톡으로 제휴(쿠팡파트너스·공동구매) 수익을 키우려는 SNS 크리에이터. 발행량을 늘리려 영상 편집·썸네일·기획을 외주로 맡기고 싶은 1인 운영자 김OO
