# 촬영몬 와이어프레임

작성일: 2026-07-06
기준 문서: `architecture.md`(라우팅/게이트/플로우), `design.md`(토큰/컴포넌트), `PRD.MD`
범위: 공개 페이지 + 인증 + 등록 폼 + 모달/상태 화면. 마이페이지는 `mypage-design.md`, 관리자는 `admin-design.md`에서 다룬다.

## 0. 읽는 법

- 와이어프레임은 배치 골격만 표현한다. 픽셀 값은 `design.md` 5장(레이아웃)과 `ui-components.md`를 따른다.
- `[버튼]` = 행동, `〈배지〉` = 상태 배지, `■` = 이미지, `…` = 반복 요소
- PC 본문 최대폭 1180px, 좌측 사이드 220px. 모바일 기준폭 360px, 좌우 패딩 16px
- 모든 페이지 공통: PC는 1장 PC 공통 레이아웃, 모바일은 1장 모바일 공통 레이아웃 안에 본문만 교체된다

### 1.3 게이트 모달 3종 (architecture.md 5장)

공통 골격: 제목 1줄 + 설명 1줄 + 주 버튼 1개 + 보조(닫기). 모바일은 바텀시트.

| 모달 | 제목 | 주 버튼 | 보조 |
| --- | --- | --- | --- |
| 로그인 유도 | 로그인이 필요합니다 | [로그인] → `/login?redirect=현재` | [회원가입] 텍스트 링크, 닫기 |
| 열람권 구매 유도 | 연락처 열람권이 필요합니다 | [열람권 구매] → `/services/order/contact-pass` | 상품 3종 가격 미리보기 행, 닫기 |
| 기업 인증 유도 | 기업 인증 후 이용할 수 있습니다 | [인증하러 가기] → `/mypage/verification` | 현재 상태 배지(미인증/검수중/반려), 닫기 |

### 1.4 신고 모달

```
신고하기
( ) 불건전 업소/위험 촬영   ( ) 다단계/피라미드성 영업
( ) 낮은 임금/부당 페이     ( ) 광고 글
( ) 잘못 기재된 연락처      ( ) 저작권/초상권 침해 우려
( ) 기타
[상세 내용 textarea — 기타 선택 시 필수]
            [취소] [신고 접수]   ← 접수 버튼은 Danger
```

접수 후: "신고가 접수되었습니다. 처리 결과는 알림으로 안내됩니다." 토스트.

### 1.5 상태 화면

- **빈 상태**: 아이콘(연한 카메라/프레임) + 문구 1줄 + 행동 버튼 1개 (design.md 10.1)
- **로딩**: 목록 = 스켈레톤 카드 6개, 상세 = 제목/이미지/본문 스켈레톤, 제출 = 버튼 내 스피너
- **404**: 로고 + "페이지를 찾을 수 없습니다" + [메인으로] — 공통 레이아웃 유지
- **권한 안내 페이지** (페이지 단위 차단 시): 게이트 모달과 동일 내용을 본문 중앙 카드로

### 3.5 공고 등록 `/jobs/new` (기업·인증완료 전용, design.md 9.1 섹션 순서)

한 페이지 세로 폼, 좌측에 섹션 앵커 내비(PC만):

1. 기본 정보 — 공고 제목*, 촬영 분야*(단일 셀렉트), 대표 이미지 업로드
2. 모집 조건 — 모집인원, 경력*, 나이, 성별, 접수기간*(마감일/상시/채용시 라디오)
3. 촬영 조건 — 촬영 형태, 담당업무, 필요 장비/스킬(체크), 촬영 조건/복리후생
4. 급여/계약 — 급여 유형*(건당/일당/월급/협의) + 금액, 고용형태*
5. 담당자 정보 — 담당자명*, 이메일*, 연락처 노출 여부
6. 상세 내용 — 서식 에디터
7. 지도/주소 — 주소 검색 + 지도 미리보기
8. 노출 상품 선택 — ☐프리미엄 ☐자동점프 (가격 표시, 선택 시 결제 플로우 연결)

## 5. 인증

### 5.1 로그인 `/login`

중앙 카드 1단 (PC 400px / 모바일 풀폭): ◉개인 ○기업 라디오 → 아이디/비밀번호 → [로그인] → ─ 또는 ─ → [카카오로 시작하기] → 하단 링크: 아이디 찾기 · 비밀번호 찾기 · 회원가입
`redirect` 파라미터 있으면 성공 후 복귀.

### 5.2 가입 유형 선택 `/signup`

```
┌개인회원──────────┐  ┌기업회원/의뢰자─────┐
│ 🎥 촬영자로 활동    │  │ 🏢 촬영자를 찾습니다  │
│ 프로필 등록·공고 지원│  │ 공고 등록·지원자 관리 │
│ [개인회원 가입]     │  │ [기업회원 가입]      │
└─────────────────┘  └──────────────────┘
이미 계정이 있다면 [로그인]
```

### 5.3 개인 가입 `/signup/personal`

약관 동의(전체/필수/선택 체크) → 아이디* / 비밀번호* / 비밀번호 확인* / 이름* / 닉네임* / 이메일* / 휴대폰 + [인증요청](데모: 즉시 성공) → [가입하기] → 완료: "프로필을 등록하면 제안을 받을 수 있습니다" [프로필 등록] [나중에]

### 5.4 기업 가입 `/signup/company`

약관 동의 → 계정 정보(아이디/비밀번호) → 기업 정보: 회사명* / 대표자명* / 사업자등록번호* / 업종 / 담당자명* / 담당자 연락처* / 이메일* → ⓘ "공고 등록에는 기업 인증(사업자등록증명, 3개월 이내)이 필요합니다. 사업자가 없는 개인 의뢰자는 본인 명의 통장 사본으로 대체할 수 있습니다." → [가입하기] → 완료: 상태 〈미인증〉 표시 + [지금 인증하기 → /mypage/verification] [나중에]

### 5.5 계정 찾기 `/account/find`

탭: 아이디 찾기(이름+이메일) | 비밀번호 재설정(아이디+이메일). 데모에서는 마스킹 결과/성공 토스트만.
