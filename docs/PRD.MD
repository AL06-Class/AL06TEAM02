## 2. 제품 개요

### 2.1 한 줄 정의

CLIPBee는 촬영 의뢰자와 촬영자/촬영팀을 연결하는 구인구직, 프로필 탐색, 커뮤니티, 스토어, 유료 노출 플랫폼이다.

### 2.2 핵심 사용자

- 비회원: 공고, 프로필, 커뮤니티, 스토어를 탐색한다.
- 개인회원: 촬영자 프로필을 등록하고, 촬영 공고에 지원하고, 포트폴리오를 관리한다.
- 기업회원/의뢰자: 촬영자 모집 공고를 등록하고, 지원자를 관리하고, 촬영자 프로필을 열람한다.
- 판매자: 스토어에 촬영 관련 서비스나 디지털 상품을 등록한다.
- 관리자: 회원, 기업인증, 공고 심사, 신고, 결제상품, 배너, 게시판, 스토어를 관리한다.

### 2.3 핵심 목표

- 촬영 구인구직 정보를 빠르게 검색하고 비교할 수 있어야 한다.
- 촬영자 프로필을 포트폴리오 중심으로 노출해야 한다.
- 의뢰자가 촬영자 연락처를 유료 또는 권한 기반으로 열람할 수 있어야 한다.
- 공고 등록, 프로필 등록, 스토어 등록은 로그인 후 가능해야 한다.
- PC와 모바일 모두 편집몬 수준의 동일 기능 접근성을 제공해야 한다.

  - 공지사항

  - 커뮤니티: 자유게시판, 피드백게시판, 공모전, 운영자에게 바란다
- 하단 PC버전 전환 버튼

### 5.1 PC 메인

구성:

- 상단 대형 광고 배너
- 로그인 박스
- 프리미엄 배너 영역
- 프리미엄 촬영자 모집 카드 그리드
- 추천 촬영자 프로필
- 커뮤니티 요약 카드: 자유게시판, 피드백게시판, AI/촬영랩 또는 촬영노하우
- 푸터

기능:

- 배너 클릭 시 외부/내부 랜딩 이동
- 공고 카드 클릭 시 상세 이동
- 추천 프로필 클릭 시 프로필 상세 이동
- 커뮤니티 더보기 클릭 시 게시판 목록 이동
- 빠른 문의 클릭 시 카카오톡 또는 상담 채널 연결

### 5.2 모바일 메인

구성:

- 커뮤니티 게시판 묶음이 먼저 노출된다.
- 자유게시판, 공모전, 피드백게시판, AI편집랩에 대응하는 촬영 분야 게시판을 카드 목록으로 보여준다.
- 추천 촬영자 프로필이 카드 그리드로 노출된다.
- PC버전 버튼과 푸터가 있다.

CLIPBee 권장 모바일 순서:

1. 촬영자 모집 최신/프리미엄
2. 추천 촬영자 프로필
3. 커뮤니티/공모전/피드백/촬영랩
4. 공지사항

편집몬은 모바일 첫 화면에서 커뮤니티 비중이 크다. CLIPBee는 초기 전환을 위해 모집/프로필을 더 위에 두는 것이 낫다.

## 6. 회원가입과 로그인

### 6.1 회원 유형

개인회원:

- 촬영자 프로필 무료 등록 및 관리
- 온라인 지원
- 촬영자 모집 검색
- 촬영자 모집 스크랩
- 포트폴리오 등록
- 스토어 상품 등록 권한

기업회원/의뢰자:

- 촬영자 모집 공고 등록
- 온라인 지원자 관리
- 촬영자에게 제안
- 촬영자 프로필 스크랩
- 촬영자 연락처 열람권 구매
- 사업자/신원 인증

### 6.2 로그인 기능

필수:

- 개인회원/기업회원 구분 로그인
- 아이디/비밀번호 로그인
- 카카오 로그인
- 아이디/비밀번호 찾기
- 회원가입 안내 진입
- 로그인 필요 페이지 접근 시 로그인 페이지로 리다이렉트

### 6.3 기업 인증

확인된 정책:

- 공고 등록 기업은 사업자등록증명이 필요하다.
- 사업자등록증명은 최근 3개월 이내 문서여야 한다.
- 회사명, 대표자, 사업자등록번호가 일치해야 한다.
- 사업자가 없는 유튜버/개인 의뢰자는 본인 명의 통장 사본 등 대체 증빙을 받을 수 있다.

CLIPBee 요구사항:

- 기업회원 가입 후 `미인증`, `검수중`, `인증완료`, `반려` 상태를 가진다.
- 인증완료 전에는 공고 등록을 제한한다.
- 관리자는 증빙 파일을 보고 승인/반려/보완요청을 할 수 있어야 한다.

## 7. 촬영자 모집

### 7.1 PC 목록

확인된 편집몬 구조:

- 좌측 카테고리: 전체, 근무형태별, 경력별
- 상단 검색 박스: 편집가능툴 체크박스, 경력, 무관포함, 키워드 검색
- 정렬: 등록일순, 수정일순, 마감일순, 조회순, 경력 오름차순, 경력 내림차순
- 목록 컬럼: 회사명, 모집제목, 가능툴, 경력, 마감일
- 공고 상태 배지: 프리미엄프로, 경력무관/연차, 채용시/상시채용/마감일
- 페이지네이션

CLIPBee PC 목록 요구사항:

- 좌측 카테고리:
  - 전체 촬영자 모집
  - 촬영 분야별: 브랜드/광고, 유튜브, 숏폼, 웨딩, 행사, 제품, 인터뷰, 라이브, 드론, 스튜디오, 부동산, 교육, 뷰티/패션
  - 지역별
  - 경력별: 신입, 1년 이상, 3년 이상, 5년 이상, 10년 이상
  - 장비별: 카메라, 렌즈, 조명, 짐벌, 드론, 오디오, 라이브 송출, 스튜디오 보유
- 상단 필터:
  - 촬영 분야 체크박스
  - 장비/스킬 체크박스
  - 경력 선택
  - 무관포함 체크
  - 키워드 검색 범위: 회사명, 모집제목, 모집내용
- 목록 카드/행:
  - 회사명/의뢰자명
  - 모집 제목
  - 촬영 분야/장비 배지
  - 경력 조건
  - 근무/촬영 지역
  - 급여/견적 기준
  - 마감일/상시/채용시
  - 프리미엄 노출 여부
- 페이지네이션은 PC 10개 번호 단위로 제공한다.

### 7.2 모바일 목록

확인된 구조:

- 상단 탭에서 촬영자 모집 진입
- 프리미엄 배너 카드가 먼저 노출된다.
- 일반 모집 리스트가 카드형으로 노출된다.
- 카드에는 회사명, 제목, 대표 이미지, 경력, 마감일이 표시된다.
- 페이지네이션은 1~5 후 다음 버튼 형태다.

CLIPBee 모바일 요구사항:

- 2열 프리미엄 배너 카드
- 1열 일반 공고 카드
- 제목은 최대 2~3줄 표시
- 배지: 프리미엄, 경력, 마감
- 필터 진입은 직종별/지역별/역세권별/상세검색으로 분리
- PC버전 버튼 유지

### 7.4 온라인 지원

요구사항:

- 로그인한 개인회원만 온라인 지원 가능
- 지원 시 등록된 프로필/이력서를 선택한다.
- 받는 사람은 기업 담당자로 자동 입력된다.
- 보낸 사람, 지원 내용, 첨부 포트폴리오를 제출한다.
- 기업회원 마이페이지에서 지원자를 확인한다.

## 23. 비기능 요구사항

- PC와 모바일 모두 동일 핵심 기능을 제공한다.
- 이미지 업로드는 썸네일 자동 생성이 필요하다.
- YouTube/Shorts 링크 미리보기를 지원한다.
- 지도는 Kakao Map 또는 Naver Map 연동이 필요하다.
- 개인정보는 비회원에게 노출하지 않는다.
- 연락처 열람 로그를 저장한다.
- 파일 업로드는 확장자/용량/악성파일 검사를 수행한다.
- 공고/프로필/스토어/게시글 신고 기능을 제공한다.
- 봇 방지 또는 자동등록방지 절차를 제공한다.
- 결제/환불 로그는 관리자 감사 로그에 남긴다.
- 관리자 변경 이력은 삭제하지 않는다.

## 24. 법무/운영 주의사항

- 직업정보제공사업 관련 법적 요건을 검토해야 한다.
- 구인자 신원 확인 프로세스는 CLIPBee에도 필수로 둔다.
- 사업자등록증명이 없는 개인 의뢰자 대체 인증 절차를 마련한다.
- 임금체불/저단가/불건전 촬영 신고 절차를 명확히 둔다.
- 포트폴리오 영상은 저작권/초상권 확인 안내를 제공한다.
- 스토어 상품은 저작권 규정과 환불 규정을 명확히 표시한다.
- 편집몬 고유 이미지, 로고, 카피, 게시글 원문은 복제하지 않는다.

## 26. 개발 수용 기준

런칭 전 반드시 확인:

- PC 메인/목록/상세/서비스/스토어 화면이 동작한다.
- 모바일 메인/목록/상세/카테고리/검색 화면이 동작한다.
- 비회원은 연락처를 볼 수 없다.
- 개인회원은 프로필 등록과 공고 지원을 할 수 있다.
- 기업회원은 인증 완료 후 공고를 등록할 수 있다.
- 관리자는 공고와 인증을 승인/반려할 수 있다.
- 자동점프는 결제 후 마이페이지에서 활성화해야 작동한다.
- 추천 프로필은 결제 기간 동안 메인에 노출된다.
- 프리미엄 배너는 결제/검수 후 지정 위치에 노출된다.
- 스토어 상품은 등록, 목록, 상세, 신고, 환불 규정 표시가 가능하다.
- 신고 접수와 관리자 처리가 가능하다.
- PC/모바일 모두 페이지네이션과 상세 이동이 정상이다.
