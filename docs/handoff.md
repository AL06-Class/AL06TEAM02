## 2. 확정 사항 요약

- **스택**: Next.js(App Router) + TypeScript + Tailwind. 백엔드 없음, `data/*.ts` + localStorage
- **스토어 1차**: 목록 + 상세 + 등록 (주문/결제/리뷰/찜 고도화는 2차)
- **모바일**: 반응형 단일 코드베이스, "PC버전" 버튼은 viewMode 강제 전환
- **인증**: 데모 역할 스위처 5역할 (비회원/개인/기업 미인증/기업 인증/관리자)
- **지도**: 1차는 정적 이미지 + 주소 텍스트
- **모든 유료 기능은 모의결제** — 결제 후 상태 변경만 시뮬레이션

## 4. 검증 체크리스트 (AGENTS.md 14장 + 수용 기준)

기능 (PRD 26장):
- [ ] 비회원은 연락처를 볼 수 없다 (프로필 상세에서 확인)
- [ ] 개인회원 역할: 프로필 등록·공고 지원 가능
- [ ] 기업 미인증 역할: 공고 등록 진입 시 인증 유도
- [ ] 기업 인증+열람권 역할: 연락처 열림 + 제안 버튼 노출
- [ ] 자동점프: 결제 완료만으로 활성화되지 않고 마이페이지 토글 필요
- [ ] 관리자 반려 → 기업 마이페이지에 반려 배너 즉시 표시
- [ ] 신고 접수 → 관리자 신고 목록에 노출

UI (design.md 18장):
- [ ] 모바일 360px에서 카드 제목 넘침 없음
- [ ] 프리미엄/추천/공지/마감 배지 색 구분 (architecture 9장 매핑)
- [ ] 빠른 문의 버튼이 본문/하단 고정 바를 가리지 않음
- [ ] 페이지네이션·카테고리 URL 유지 (뒤로가기 시 필터 보존)
- [ ] 모든 이미지 alt 존재, 아이콘 버튼 aria-label 존재

캡처 검증:
- [ ] PC/모바일 주요 화면 Playwright 캡처 → `output/playwright/shootmon-*` 저장
- [ ] 원본 편집몬 자산(로고/이미지/문구/사업자 정보) 미포함 최종 확인
